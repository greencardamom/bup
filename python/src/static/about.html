BooksUp

About

BooksUp adds archive.org book and journal links to citations on the English Wikipedia, so readers can jump straight to the cited page of a scanned source.

Articles are pre-processed offline to match citations with digitized books and journals; each proposed citation is stored and applied only after a logged-in editor reviews it, with the edit attributed to that editor. Because articles change, the number of links actually added may be fewer than listed if a citation was edited since it was processed.

See also the on-wiki BooksUp gadget, which surfaces the same suggestions while you edit a page.
